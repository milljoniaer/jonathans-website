# Jonathan's Personal Website

This is my personal portfolio website showcasing my professional experience, projects, skills, and writing.

## About This Website

This is a modern, responsive portfolio website built to showcase my work and provide information about my professional background. 

## Development

To run this website locally:

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

This project is built with modern web technologies:

- **Vite**: Fast build tool and development server
- **TypeScript**: Type-safe JavaScript
- **React**: UI component library
- **shadcn/ui**: Beautiful and accessible UI components
- **Tailwind CSS**: Utility-first CSS framework
- **Lovable**: AI generation of whole WPA's. Its insane!!! Check it out!
